## The format

Every concept is one Markdown file with a small YAML frontmatter header:

```yaml
---
type: officials
title: Babajide Sanwo-Olu
description: Governor of Lagos State (APC)
resource: https://ournigeria.ng/officials/babajide-sanwo-olu
tags: [lagos, apc, governor]
timestamp: 2026-06-29T00:00:00Z
---
```

- **`type`, `title`, `description`, `resource`, `tags`, `timestamp`** are the queryable fields. `resource` is the canonical page on the live site.
- The body holds the prose and structured sections.
- Concepts cross-link with ordinary Markdown links, e.g. `[Lagos](/states/lagos.md)`, forming a graph rather than a flat list. States and parties accumulate inbound links, so their pages act as live indexes.

## Provenance

Every sourced fact renders a footnote: the publisher, a short quote, the source tier, the date retrieved, and — where a takedown-proof copy was captured — an **archived snapshot** link. If an original source goes offline, the document says so and the archived copy becomes the reference. The point is civic claims you can trace back to where they came from.
