# OurNigeria — Public Knowledge Bundle

This repository is an **auto-generated** [Open Knowledge Format (OKF)](https://github.com/GoogleCloudPlatform/knowledge-catalog/tree/main/okf) bundle of OurNigeria's civic data — Nigerian public officials, corruption cases, political parties, states and local government areas — as portable, cross-linked markdown with cited sources.

- **Browse the graph:** open `viz.html`
- **Start here:** [`index.md`](./index.md)
- **Source project:** https://github.com/LogicalOgbonna/ournigeria · https://app.ournigeria.ng

The bundle is rebuilt and pushed automatically. Each fact links to its source; where the original may disappear, an archived snapshot copy is linked too.

_Do not edit by hand — changes are overwritten on the next publish._
